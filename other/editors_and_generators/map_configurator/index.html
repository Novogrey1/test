Конфигуратор опубликованной карты

Параметр Name — внутреннее уникальное имя карты в системе игры. Рекомендуется использовать только латинские буквы и цифры.

Пожалуйста, введите название карты.

Параметр Image — числовой ID изображения-превью карты, загруженного в Roblox. Как получить ID:
1. Зайдите на create.roblox.com
2. Откройте Creations → Development Items → Decals / Images
3. Загрузите нужное изображение и скопируйте числовой ID из карточки ассета или адресной строки
Поле необязательное — если оставить пустым, изображение для карты задано не будет.

ID изображения должен быть положительным числом.

Управляет загрузкой карты по умолчанию. Если true — стандартная карта не загружается. Значение обязано совпадать с OnLaunchNotLoadDefaultMap на сервере — несоответствие вызовет ошибку загрузки.

Аналог параметра OnLaunchNotLoadDefaultMapSpawnsAndRoutes. Если true — маршруты и остановки по умолчанию не загружаются. Для полностью кастомных карт без стандартной разметки маршрутов.

Инструкция по использованию редактора карт

Принцип работы конфигуратора карт в Ved's Trolleybus Place. Внимательно ознакомьтесь с инструкцией во избежание недопонимания.

  • Название карты (Name) — уникальный внутренний идентификатор карты. Рекомендуется только латиница и цифры. (Обязательный параметр)
  • ID изображения (Image) — числовой ID превью-картинки из Roblox. Загрузите изображение на create.roblox.com, затем найдите ассет в разделе «Creations → Development Items» и скопируйте ID из карточки.
  • IsEmpty — определяет, загружается ли стандартная карта при старте. При значении true стандартная карта не загружается. Должно совпадать с OnLaunchNotLoadDefaultMap.
  • IsEmptyRoutesAndStops — определяет, загружаются ли маршруты и остановки по умолчанию. Аналог настройки OnLaunchNotLoadDefaultMapSpawnsAndRoutes.

Примечание: Для применения конфигурации вставьте сгенерированный JSON в TrPEditMapDetails.
